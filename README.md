少年，你可曾想过

行侠仗义，锄强扶弱？

少年，你可曾希冀

匡正天下，还道于世？

少年，你和你的同伴

那份友谊

是否久久萦绕脑海，未曾忘怀？

我们的幻想

纵然不能变成现实

也权且让它

在梦里实现一回罢！

我愿以这份游戏，作为一份回忆。

亦是，一份赠礼罢。



——IsaacZhang
